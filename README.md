# Web Store Project

## 📌 Overview

A simple e-commerce web application developed using HTML, CSS, and JavaScript. The project simulates an online shopping experience with product browsing, shopping cart, seller page, and user profile.

---

## 🚀 Features

- Product catalog
- Shopping cart
- User profile
- Seller dashboard
- Responsive interface
- Client-side interaction using JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript

---

## 📂 Project Structure

- HTML Pages
- CSS Stylesheets
- JavaScript Files
- Images
